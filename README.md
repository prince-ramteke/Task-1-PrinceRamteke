# Number Guessing Game 🎯

## Project Information
This project was developed as part of the DecodeLabs Industrial Training Kit - Java Programming Project 1.

## Objective
Create a Java program that:

- Generates a random number between 1 and 100
- Accepts user guesses
- Provides feedback if the guess is too high or too low
- Continues until the correct number is guessed

## Features
✅ Random number generation using Random class

✅ User input using Scanner class

✅ High / Low feedback system

✅ Input validation using Exception Handling

✅ Limited attempts (7 attempts)

✅ Score tracking

✅ Play Again functionality

## Technologies Used

- Java
- Random Class
- Scanner Class
- Loops
- Conditional Statements
- Exception Handling
